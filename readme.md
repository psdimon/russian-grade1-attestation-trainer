# РусЯз 1 класс: тренажер к аттестации

Веб-проект для подготовки учеников первого класса к аттестации по русскому языку.

## Демо

  GitHub Pages: https://psdimon.github.io/russian-grade1-attestation-trainer/

## Возможности

- Генерация вариантов заданий по seed.
- Режим бланка и режим с ответами.
- Печать готового варианта.
- Production-сборка для статического хостинга.

## Технологии

- `Vite`
- `JavaScript`
- `HTML/CSS`

## Запуск локально

```bash
npm install
npm run dev
```

## Production-сборка

```bash
npm run build
```

Результат сборки: файлы в директории `dist/` (`index.html` и ассеты в `dist/assets/`).
